***General introduction***
This dataset contains data collected during document analysis at the Vrije Universiteit Amsterdam, as part of Sanne Groothuis' PhD research project (Spring 2025). Doi to be published after defense.
It is being made public both to act as supplementary data for publications and Sanne Groothuis' PhD thesis and in order for other researchers to use this data in their own work. 

The data in this data set was collected online, produced and/or analysed between 2021 and 2022. This research project was made possible by the Vrije Universiteit Amsterdam. 

***Purpose of the data collection and analysis***
The purpose of this data collection and analysis was to investigate patterns of racialisation in Dutch counter-radicalisation policies. 

***Description of the data in this data set***
The data included in this data set has been organised per data type. The files follow the nomenclature system: X_data_Y_Z with
X = the status of the data (raw, processed, analysed)
Y = description of the data
Z = format 

It includes the code list, list of analysed policy documents, interview topic list, and coded policy documents.
